# 项目范围

## 基本信息

- 项目名称：药研制策
- 英文名称：Pharma R&D Decision Agent
- 当前拟定方向：面向非小细胞肺癌的多源证据驱动研发管线分析
- 疾病：非小细胞肺癌（NSCLC）
- 企业：恒瑞医药、百济神州（BeOne Medicines，原BeiGene）

## 初版范围

- 每家 3-5 个代表性药物
- 总计 10-20 条临床试验
- 总计 10-20 篇论文
- 总计 5-10 条监管或企业正式资料

## 核心功能

- 研发管线比较
- 多源证据检索
- 同类管线比较
- 证据追溯
- 风险提示
- 带引用报告

核心用途：研发管线梳理、多源证据检索、同类管线比较、证据追溯和风险提示。

## 明确不做

- 疾病诊断或诊断建议
- 个体治疗建议或用药建议
- 药物疗效结论或疗效判断
- 投资建议
- 全量医药数据库
- 模型训练
- 复杂知识图谱和多智能体

本项目不提供疾病诊断、个体治疗建议、药物疗效结论或投资建议。

## 当前状态

恒瑞医药三批资料核验完成，累计 15 条；百济神州第一、第二批资料核验完成，累计来源记录 11 条。两家企业均已发现可用 NSCLC 资料，跨企业比较初步可行，但仍需完成百济神州管线及正式公告核验，并补充 NMPA/CDE 等独立监管数据库原始记录。来源记录数与唯一临床试验数需分开统计。
