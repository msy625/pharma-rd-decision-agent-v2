# 项目范围

## 基本信息

- 项目名称：药研制策
- 英文名称：Pharma R&D Decision Agent
- 当前拟定方向：面向非小细胞肺癌的多源证据驱动研发管线分析
- 适应症：非小细胞肺癌（NSCLC）
- 比较企业：恒瑞医药、百济神州

## 初版范围

- 每家 3-5 个代表性药物
- 总计 10-20 条临床试验
- 总计 10-20 篇论文
- 总计 5-10 条监管或企业正式资料

## 核心功能

- 研发管线比较
- 证据检索
- 证据追溯
- 风险提示
- 带引用报告

## 明确不做

- 诊断建议
- 用药建议
- 疗效判断
- 投资建议
- 全量医药数据库
- 模型训练
- 复杂知识图谱和多智能体

## 当前状态

方向拟定，待完成数据可用性检查后正式冻结。
