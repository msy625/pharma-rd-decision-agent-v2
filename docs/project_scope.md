# 项目范围

## 基本信息

- 项目名称：药研制策
- 英文名称：Pharma R&D Decision Agent
- 当前拟定方向：面向非小细胞肺癌的多源证据驱动研发管线分析
- 疾病：非小细胞肺癌（NSCLC）
- 企业：恒瑞医药、百济神州（BeOne Medicines，原BeiGene）

## 初版范围

- 每家 3-5 个代表性药物
- 总计 10-20 条临床试验
- 总计 10-20 篇论文
- 总计 5-10 条监管或企业正式资料

## 核心功能

- 研发管线比较
- 多源证据检索
- 同类管线比较
- 证据追溯
- 风险提示
- 带引用报告

核心用途：研发管线梳理、多源证据检索、同类管线比较、证据追溯和风险提示。

## 明确不做

- 疾病诊断或诊断建议
- 个体治疗建议或用药建议
- 药物疗效结论或疗效判断
- 投资建议
- 全量医药数据库
- 模型训练
- 复杂知识图谱和多智能体

本项目不提供疾病诊断、个体治疗建议、药物疗效结论或投资建议。

## 当前状态

第一版范围已冻结。疾病范围为 NSCLC，比较企业为恒瑞医药与百济神州（BeOne Medicines，原BeiGene）。后续新增资料进入补充版本，不再影响初版开发主链路。

恒瑞医药与百济神州的第一版数据可用性检查已经完成。现有 31 条人工核验来源覆盖公司官网、年度报告、ClinicalTrials.gov、PubMed 论文、公司管线材料、公司正式公告及 EMA 独立监管资料，能够支持 NSCLC 研发管线、试验阶段、研究人群、证据版本和监管状态的初版比较。

保留限制：恒瑞仍可补充 NMPA/CDE 独立监管原始记录；两家公司来源类型并非完全对称；公司管线图不能单独作为历史研究状态依据；论文数量不能作为唯一试验数量；项目不提供诊断、个体治疗建议、疗效保证或投资建议。
