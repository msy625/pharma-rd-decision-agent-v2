# 第三天集成验证报告

## 第三天目标

第三天目标是在当前冻结的 31 条已人工核验 NSCLC 资料基础上，完成证据链关系、证据链服务/API/Web 页面，以及企业证据对比服务/API/Web 页面，使“研发证据查询”从单条来源检索扩展为可追溯的证据链和当前样本对比。

本阶段不调用大模型、Chroma 或向量模型，不生成评分、排名、成功率、疗效排名或投资建议。

## 证据链范围

当前采用人工确认配置 `config/evidence_chains.json`，不使用标题相似度或自动推断建立临床研究关系。

- 试验级证据链：10 条。
- 药物级监管链：1 条。
- 合计：11 条。

药物级监管链不计入临床试验数量。同一试验下的登记、论文、公司页面和监管背景不能分别重复计为多项试验。

## 配置、服务、API 和 Web 调用关系

```text
data/source_registry.csv
          ↓
SourceRegistryService
          ↓
config/evidence_chains.json → EvidenceChainService
          ↓
FastAPI /api/evidence/chain-*
          ↓

```

企业证据对比调用关系：

```text

          ↓

```

## RATIONALE-315 与 B016

RATIONALE-315 / NCT04379635 试验级证据链只包含：

- B011：中期论文。

B016 可作为 RATIONALE-315 的关联监管背景展示，但不进入 NCT04379635 的试验证据数量，不增加该试验链 `source_count`。

## B015/B016 监管状态

药物级监管链 `regulatory:tevimbra-eu-nsclc` 包含 B015 和 B016。

- B015：EMA 正式授权信息，授权状态为“欧盟正式授权”。
- B016：CHMP 积极意见，授权状态展示为“积极意见，非最终批准”。

B016 不显示为正式授权、正式批准或最终批准。

## 对比限制和禁止结论

当前禁止输出：

- 企业综合评分。
- 企业排名、优胜方或“领先”判断。
- 成功率预测。
- 跨试验疗效或安全性优劣比较。
- 投资建议。
- 从标题或药物名称自动推断研究人群、治疗场景、靶点、机制或药物类型。

临床阶段、研究人群、治疗场景、靶点、机制、药物类型、疗效与安全性在当前版本均属于“当前数据不足”或“只能部分比较”的维度。

## 当前限制

- 当前数据范围仍限于第一版 31 条已人工核验 NSCLC 资料。
- 恒瑞部分 PubMed 论文与具体登记试验的一对一关系仍待补充 NCT 或 parent_trial_id。
- H014 属于公司/药物级资料，当前无法确认其与具体临床试验的一对一关系。
- B015 是药物级正式授权资料，但与具体 RATIONALE 试验的一对一关系仍需 EPAR 评审报告进一步核验。
- 企业证据对比不覆盖完整企业管线、真实药物项目总数、靶点机制全景或跨试验疗效结论。
